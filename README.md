## 규칙
- **화, 목 9시** 모임 (웹엑스 링크 예정)
- **화요일**
  - 문제 리뷰 및 목요일 문제 공유
- **목요일**
  - 문제 리뷰 및 화요일 문제 공유
- 문제는 각자 안풀은 것 or 실패한 것 위주로 구성
